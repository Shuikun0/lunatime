# Lunatime

*A mostly harmless guide to poking a planet and asking how long the big bright thing will keep staring.*

在地球上点一下，立刻看到：

- **日照时长**、日出 / 日落
- **太阳高度**（当天曲线）
- **紫外线指数**估算（含相对安全时段）
- 时区、日期、时刻可调；视角与选择会记住

单文件 `index.html`。无后端，无账号，无「请先同意条款再仰望苍穹」。

## 怎么开

浏览器打开 [`index.html`](./index.html)，或：

```bash
python3 -m http.server 8780
```

→ http://localhost:8780  

macOS 也可双击 `启动 Lunatime.command`。

## 关于数字

教育用近似值——够建立直觉，不够替你决定是否涂防晒。别拿去打官司；恒星不会出庭。
