# Lunatime

*A mostly harmless guide to standing on a spinning rock and asking how long the big bright thing will keep looking at you.*

在银河系的某个不起眼角落，有一颗蓝绿色的行星。它的居民发明了农业、诗歌、以及一种更危险的东西：对「现在几点」的执着。

Lunatime 是一个交互式地球日照 / UV 玩具。你在球面上点一下，它就尽量礼貌地告诉你：日照大概多久、太阳大概多高、紫外线大概多凶。它不提供人生建议，也不提供毛巾¹。

¹ 毛巾请自备。宇宙不负责。

## 怎么启动

任选其一。宇宙对方法不挑剔，只对结果偶尔刻薄。

- 用浏览器打开 [`index.html`](./index.html)
- 或：

```bash
python3 -m http.server 8780
```

然后访问 http://localhost:8780

macOS 用户也可双击 `启动 Lunatime.command`。这通常比向本地环回地址解释存在主义更高效。

## 关于那些数字

它们是教育用的近似值——足够让你对日照与紫外线产生一种体面的直觉，却不足以替你决定是否涂防晒、是否出门、或是否继续与恒星保持目前这种单方面的关系。

换言之：别拿它去打官司。恒星不会出庭。
