# Lunatime

地球太大，太阳太远——所以我们做了个能点的。

Lunatime 是一个交互式地球日照 / UV 演示：在球面上戳一下，就能看到日照时长、日出日落、太阳高度，以及紫外线指数的估算。单文件 `index.html`，没有后端，没有账号，没有「请先登录再晒太阳」。

## 怎么开

浏览器直接打开 [`index.html`](./index.html)，或本地起个静态服务：

```bash
python3 -m http.server 8780
```

然后访问 http://localhost:8780

macOS 懒人通道：双击 `启动 Lunatime.command`。

## 数字怎么看

日照与太阳高度是简化天文模型（时差方程之类的细节先放一边）。UV 指数按晴空估算，**仅供教育参考**——别拿它当出门防晒的官方判决书。
